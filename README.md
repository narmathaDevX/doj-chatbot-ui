# AI Legal Chatbot

## 📌 Description
AI-powered chatbot that provides legal information using NLP and a structured knowledge base.

## 🚀 Features
- Natural language query handling  
- Flask backend API  
- NLP-based intent recognition  
- Knowledge base retrieval  
- Privacy-focused system  

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python, Flask  
- NLP: NLTK, spaCy  
- AI: RAG + Ollama  
- Database: JSON  

## 🏗 Architecture
Layered architecture:
Frontend → Backend → NLP → Knowledge Base  

## ▶ Run Locally
pip install -r requirements.txt  
python app.py  

## ⚠ Disclaimer
This chatbot provides legal information, not legal advice.
